PCO Service Dashboard Setup
===========================

This repo supports two run modes:

1. Electron desktop app
2. Standalone Node/Express server


Electron desktop app
--------------------

Development:
  npm install
  npm run electron

Packaged build:
  npm run dist

Quick unpacked Windows build:
  npm exec electron-builder -- --dir

The desktop app opens directly to the dashboard window and uses an in-app setup wizard on first launch.


Standalone server mode
----------------------

Use this for Render deployment or browser-only local development.

1. Copy .env.example to .env
2. Fill in:
     PCO_APP_ID=...
     PCO_SECRET=...
3. Install dependencies:
     npm install
4. Start the server:
     node server.js
5. Open:
     http://127.0.0.1:3000


Render deployment
-----------------

- Runtime: Node
- Build command: npm install
- Start command: node server.js

Environment variables:
- PCO_APP_ID
- PCO_SECRET

Hosted mode still works, but the new configuration flow is optimized for the desktop app. If hosted storage is not persistent, some non-secret settings may reset after redeploys.
